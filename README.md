
# Nodejs quizzes
Wanna play a quiz?
[https://nodejs-quizzes.herokuapp.com/]

# Dev notes
## Preinstall
* [git](https://git-scm.com/downloads)
* [npm](https://nodejs.org/en/download/)

## Get repo
* git clone https://github.com/sylwolveryn/nodejs-quizzes.git

## Install packages
* cd nodejs-quizzes
* npm install
* npm install mocha -g

## Run tests
* npm test
npm test is a shortcut for mocha

## Start server
* npm start
